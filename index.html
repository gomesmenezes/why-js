O que eu acho que é JavaScript?

JavaScript é uma linguagem de programação onde podemos manipular o html e css e os elementos de uma forma que não deixe o site de uma forma estática, onde temos varias funcionalidades para deixar o site dinamico e de uma forma onde podemos utilizar para estilizar e manipular elementos

O que eu espero dessa competencia de JavaScript?

Espero dominar a manipulação de JavaScript utilizando o DOM para deixar o site dinamico, trabalhar de uma forma eficiente com JavaScript onde terei dominio para utilizar ele.
